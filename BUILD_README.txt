========================================
BUILD EXE INSTRUCTIONS
========================================

วิธี Build โปรแกรมเป็น EXE ไฟล์

----------------------------------------
วิธีที่ 1: ใช้ Python Script (แนะนำ)
----------------------------------------
1. เปิด Command Prompt หรือ PowerShell
2. รันคำสั่ง:
   python build.py

----------------------------------------
วิธีที่ 2: ใช้ Batch File (เต็มรูปแบบ)
----------------------------------------
1. ดับเบิ้ลคลิกไฟล์ build.bat
2. รอจนกว่าจะเสร็จ

----------------------------------------
วิธีที่ 3: ใช้ Simple Build (เร็วที่สุด)
----------------------------------------
1. ดับเบิ้ลคลิกไฟล์ build-simple.bat
2. รอจนกว่าจะเสร็จ

----------------------------------------
ไฟล์ที่ได้หลัง Build
----------------------------------------
- ไฟล์ exe จะอยู่ในโฟลเดอร์ dist/
- ชื่อไฟล์: REVDownloader.exe

----------------------------------------
หมายเหตุ
----------------------------------------
- ต้องติดตั้ง Python 3.10+ และ pip ก่อน
- ต้องติดตั้ง PyInstaller (pip install pyinstaller)
- ควรใช้ virtual environment (venv)
- ขนาดไฟล์ exe ประมาณ 30-60 MB
- Windows Defender อาจแจ้งเตือนในครั้งแรก (เป็นเรื่องปกติ)
